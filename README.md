# 결혼식 예산 계산기

서울·경기 기준으로 결혼식(예식) 예산을 슬라이더로 맞춰보는 단일 HTML 페이지입니다.
혼수·신혼여행은 제외한 **순수 예식 비용** 계산기예요.

## 바로가기

👉 **https://jin02.github.io/wedding-budget-calculator/**

## 특징

- 하객 수·1인당 식대·1인당 축의금부터 예식장 / 스드메·촬영 / 예물·예단 / 기타까지 세부 항목을 슬라이더로 실시간 계산
- **총 지출 / 예상 축의금 수입 / 순 부담액(총 지출 − 축의금)** 을 한눈에 표시
- **목표 예산(1,500~3,500만 원)** 을 직접 설정하면 목표 대비 여유/초과를 알려줌
- **현실 반영 옵션** — 보증 인원(식대 = 참석/보증 중 큰 값), 축의금 귀속 비율(부모님 몫 제외), 부가세(VAT 10%) 토글, 웨딩플래너 비용
- **빠른 설정(지역·시기·시간대) 프리셋** — 지역(강남 / 서울 그 외 / 경기 판교·과천 등) · 1~12월(성수기/비수기) · 요일·시간대를 고르면 식대·예식장·스드메가 그 평균으로 자동 설정
- 하객 1명당 식대 vs 축의금 인사이트 제공 (하객이 늘수록 부담이 늘지/줄지)
- 설정을 **링크 또는 코드로 복사**해 메신저로 공유 — 파일 없이 복붙으로 서로 같은 화면을 맞출 수 있어요
- 라이트/다크 모드 자동 감지, 모바일 반응형
- 외부 라이브러리 없는 바닐라 HTML/CSS/JS, `index.html` 단일 파일

## 사용법

1. 페이지를 열고 (필요하면 **빠른 설정**에서 지역·시기를 고른 뒤) 슬라이더를 우리 상황에 맞게 조절합니다.
2. "설정 공유"에서 **링크 복사**(추천) 또는 **코드 복사**로 현재 설정을 메신저에 보냅니다.
3. 상대가 링크를 누르면(또는 받은 코드를 붙여넣고 적용하면) 똑같은 화면이 열립니다.

## 기본값·프리셋 근거 (조사 자료)

모든 기본값과 프리셋은 아래 **최근(2025년 기준) 공개 자료**를 바탕으로 산출했습니다. 추정이 들어간 부분은 그대로 밝혀 두었으니, 실제 견적이 나오면 슬라이더로 덮어쓰면 됩니다.

### 항목 기본값 (서울·경기 평균)

- **1인당 식대**: 서울 강남 8.3만 / 서울(그 외) 7.2만 / 경기 6.2만 원 (한국소비자원 참가격, 2025.6)
- **1인당 축의금**: 2025년 평균이 사상 처음 10만 원을 넘어섰고 서울이 가장 높음 → 기본 10만 원
- **스드메 패키지 중간가**: 강남 300 / 서울(그 외) 264 / 전국 292만 원. 실제로는 헬퍼비·드레스 추가·얼리스타트 등 *추가금*이 더해져 더 높아짐
- **예물**: 평균 300~500만 원 / **예단·한복**: 생략·간소화 추세
- **하객 수**: 일반 웨딩 기준 약 200명 (50명대 스몰웨딩도 증가)
- **기타(청첩장·사회·축가·폐백·예비비 등)**: 전체 100~200만 원대

> 부케·헬퍼비 등은 스드메 *추가금*에 포함되는 항목이라 별도 슬라이더로 두지 않고 스드메에 합산했습니다. 답례품은 식대가 사실상 답례라 단독 항목 대신 "예비비·기타"로 통합했습니다.

### 시기(시즌) 프리셋

- **성수기**: 3~6월(봄), 9~11월(가을) / **비수기**: 7~8월(여름), 12~2월(겨울)
- 1~12월 전체를 제공하며, 각 월은 위 시즌으로 분류됩니다 (성수기 달끼리는 값이 같고, 비수기 달은 할인 적용)
- 비수기 가격 차이(자료): 대관료 10~20%↓, 1인당 식대 5천~1만 원↓, 스드메 할인 (200명 기준 식대만 100~200만 원, 전체 300~500만 원 절약)
- **적용 보정(비수기)**: 식대 −1만 원, 대관료 −15%, 스드메 −10%

> 월 단위의 정확한 평균 단가 공개 데이터는 없어, 위 *시즌 차이*를 근거로 같은 시즌의 달은 동일하게, 비수기 달만 할인하는 시즌 모델로 근사했습니다.

### 지역 프리셋 (성수기 기준 — 식대 / 예식장 / 스드메)

| 지역 | 1인당 식대 | 대관료+꽃·연출 | 스드메 |
|---|---|---|---|
| 서울 강남 | 8.5만 | 550만 | 550만 |
| 서울(그 외) | 7만 | 400만 | 450만 |
| 경기(판교·과천 등) | 6.5만 | 350만 | 400만 |

> 식대·스드메는 소비자원 지역 중간가에 추가금을 더해 반영했고, 대관료+꽃·연출은 지역 등급에 비례한 추정치입니다.

### 시간대(요일) 프리셋 (대관료에 반영)

- **토요일 오후(프라임)** 가 가장 비쌈 → 기준(0%)
- **일요일 −15% / 평일·금요일 저녁 −20% / 비선호 시간대(일요일 오전 등) −30%**
- 자료: 금요일 저녁·일요일 오전은 토요일 오후 대비 20~30%, 평일은 10~20% 저렴

> 시간대는 주로 대관료에 영향을 주는 요소라 **대관료(+꽃·연출)에만** 반영했고, 지역·시즌 할인과 곱해져 적용됩니다. (50만 원 단위로 반올림되어, 기준값이 낮으면 −20%·−30%가 같은 값으로 보일 수 있어요.)

### 현실 반영 변수

단가뿐 아니라 금액을 크게 흔드는 구조적 변수도 반영했습니다.

- **보증 인원**: 웨딩홀은 최소 보장 식수를 계약하고, 실제 참석이 적어도 그만큼 식대를 냅니다. 식대 = `max(참석, 보증인원) × 1인당 식대`로 계산하고, 미달 시 빈자리 식대를 경고로 알려줘요.
- **축의금 귀속 비율**: 부모님 하객의 축의금은 보통 부모님께 갑니다. "우리가 받는 비율(%)"만 수입으로 잡아 순 부담액에 반영해요. (기본 100%)
- **부가세(VAT 10%) 토글**: 스드메·예식장 견적은 보통 부가세 별도라, 켜면 총 지출에 10%를 더해 실제 결제액에 가깝게 봅니다. (항목별 과세 여부는 다를 수 있는 대략적 가산)
- **웨딩플래너**: 직접 준비하면 0, 유료 플래너 이용 시 비용을 더합니다.

> 혼수(가전·가구)·신혼여행·신혼집은 "순수 예식 비용" 범위에서 의도적으로 제외했습니다.
